 # SmartCart E-commerce Project

## 📌 Project Description
This is a full-stack e-commerce web application built using Django and React, later upgraded to Next.js.

## 🛠️ Tech Stack
- Backend: Django + Django REST Framework
- Frontend: React (Phase 1) → Next.js (Phase 2)
- Database: PostgreSQL

- Day 1: Django project setup and initial server run

- Day 3: Setup React frontend and fetched products from Django API
